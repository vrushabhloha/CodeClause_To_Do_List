# To-Do List Project

This is a simple to-do list project that helps you keep track of your tasks and manage your daily activities.

## Features

- Add tasks: You can add new tasks to the to-do list.
- Mark tasks as complete: Once you've completed a task, you can mark it as complete.
- Remove tasks: You can remove tasks from the list once they are no longer needed.
- View tasks: You can view the list of tasks along with their status (complete/incomplete).
